## Quick start

- Clone the repo: `git clone https://github.com/Zeemzo/jigsaw-client.git`.
- Enter Directory : `cd jigsaw-client`.
- Install dependencies : `npm install`.
- Run app: `npm run start`.
- Build app: `npm run build`.
